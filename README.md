# Sentiment Analysis Project

## 📌 Overview
This project is part of the **CodeAlpha Data Analytics Internship**. It performs sentiment analysis on text reviews using Natural Language Processing (NLP) techniques. The goal is to classify customer reviews into Positive, Negative, and Neutral sentiments.

## 🚀 Features
- Data preprocessing of text reviews
- Sentiment analysis using TextBlob
- Classification into Positive, Negative, and Neutral
- Visualization of sentiment distribution

## 🛠️ Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib / Seaborn
- Natural Language Processing (NLP)

## 📊 Project Workflow
1. Import and load dataset
2. Clean and preprocess text data
3. Perform sentiment analysis using TextBlob
4. Classify sentiments (Positive, Negative, Neutral)
5. Visualize results using graphs

## 📈 Results
The analysis shows that most reviews are positive, indicating overall customer satisfaction.

## 📌 Conclusion
This project demonstrates how data analytics techniques combined with NLP can be used to analyze customer feedback and extract meaningful insights.
